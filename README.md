# Swifty

A simple Swift programming language reviewer app.

## Features

- **Timed Quizzes:** Test your Swift knowledge under pressure with a built-in countdown timer on every question.
- **Core Concepts:** Review multiple-choice questions covering essential Swift language features and syntax.
- **Instant Results:** Get immediate, detailed feedback on your score and overall performance at the end of each quiz.
- **Modern Design:** The app uses a clean, card-based interface focused on readability and a smooth user experience.
- **Dark Mode Support:** Enjoy comfortable reading in any environment thanks to full support for both light and dark themes.

## Installation

You can download the latest **Android APK** file and install the application manually from the **[releases page](https://github.com/kurtpetrola/Swifty/releases)**.
